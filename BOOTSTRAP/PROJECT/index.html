دورة HTML

HTML هي طريقة لكتابة وهيكلة صفحات الويب، ولاتعتبر HTML لغة برمجية بالمعنى الفعلي بل هي لغة وصفية أي أنها تستخدم لوصف الصفحة وترتيب وتوزيع محتواها من نصوص وروابط وصور ومقاطع فيديو وما إلى ذلك، وهي المكون الرئيسي لأي موقع ويب. تستخدم HTML مايسمى بالوسوم او tags وهي عبارة عن رموز لكل منها مهمة خاصة يقوم بها, فمنها مايقوم بعرض العنوان ومنها مايقوم بعرض النصوص، ومنها مايقوم بعرض الصور أو الروابط او أو مقاطع الفيديو وهكذا. والمتصفح يفهم هذه الوسوم أو الرموزويعرضها بالشكل المحدد أو المطلوب.

أهداف هذه الدورة

  • التعرف على مفهوم HTML
  • التعرف على كل tag والغرض منه
  • طرق إضافة النصوص، الفيديوهات، وغيرها من المحتويات المعتادة في صفحة الويب

متطلبات هذه الدورة

  • متصفح (سفاري، قوقل كروم)
  • محرر نصوص
