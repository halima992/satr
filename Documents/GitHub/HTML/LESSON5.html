HTML هي طريقة لكتابة وهيكلة صفحات الويب, ولاتعتبر HTML لغة برمجية بالمعنى الفعلي بل هي لغة وصفية أي أنها تستخدم لوصف الصفحة وترتيب وتوزيع محتواها من نصوص وروابط وصور ومقاطع فيديو وما إلى ذلك, وهي المكون الرئيسي لأي موقع ويب. تستخدم HTML مايسمى بالوسوم او tags وهي عبارة عن رموز لكل منها مهمة خاصة يقوم بها,
